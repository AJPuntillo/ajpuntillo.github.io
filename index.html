  • Aeon Engine / Archon

    The Aeon Engine is a custom engine that I built using C++ for my final year capstone at Humber College. Using OpenGL, I created a framework that contains rendering, shaders, model loading, lighting, input management, and more.

    Using this framework, I developed Archon, a 3D isometric turn-based strategy game. Throughout the development of the engine and the game, I learned a lot about the power of C++ and being able to optimize more efficiently. I also learned a lot about graphical programming and enjoying developing shaders for the game.

    Check out the code on Github

  • Steel Dungeon

    Steel Dungeon is a 2D roguelike dungeon crawler made in Unity. The game consists of the player battling through randomly chosen maps, collecting loot, and becoming more powerful as they advance. Items, enemies, and bosses are randomly spawned for each map. The game a has been developed for both PC and mobile devices.

    Responsibilites: level design, AI of the enemies, level generation, RNG, and the menus.

    Check out the code on Github

  • Creature Fighter

    Creature Fighter is a 3D action RPG made in Unreal. Players must work together, hacking and slashing to defeat boss monsters and gain loot. The game is played with up to 4 players and networked. The boss scales up in power the more players in the game.

    Responsibilites: level design, networking, UI/menus, and gameplay.
